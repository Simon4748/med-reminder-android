# Medication Reminder Application (Android)

A medication reminder app for Android, built with Kotlin and Jetpack Compose.

## Purpose
This project is focused on clean architecture, reliable reminders, and elderly-friendly UX.  
It does not provide medical advice or dosage recommendations.

## Status
Early development, just Android native for now.

## Tech Stack
- Kotlin
- Jetpack Compose
- Room for local persistence
- Hilt
- Coroutines + Flow
- ML Kit Text Recognition (OCR, on-device)
- WorkManager/AlarmManager strategy for reminders

## Design Principles
- Large text, high contrast, minimal navigation.
- Offline-first and privacy-focused.
- Clear disclaimers for OCR accuracy and no medical claims.
  
## Explicit Non-Goals
- No dosage recommendations or medical advice.
- No drug interaction checks or health outcome claims.


## Notes
This project is under active development and not intended for production medical use. This project's primary purpose is a portfolio application.
